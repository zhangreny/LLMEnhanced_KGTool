### 知识图谱生成与应用系统——功能设计

##### 一、领域名层功能

- [x] 数据库鉴权信息的编辑和连接
- [x] 获取数据库中的所有领域并罗列展示，领域名前有图标，并存在新建领域按钮
- [x] 点击领域，可展开显示领域内所有知识维度，知识维度前有图标，并存在新建维度按钮

##### 二、维度层功能

- [x] 点击维度，右侧展示此维度下的所有分类，如果没有分类，显示暂无维度分类。如果有分类，下面也有素材，便像资源管理器那样展示树状图。
- [ ] 维度下分类可在树状图上通过右键实现快速的添加，删除，移动
- [ ] 删除某维度，编辑维度名称
- [ ] 点击维度，右边可直接选展示素材列表，并提供排序功能
